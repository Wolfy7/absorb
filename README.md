# Installation

To install `absorb`, run this command in your terminal:

```
pip install absorb
```

# Usage

`absorb`'s usage looks like:

```
absorb [OPTIONS]
```

`absorb` provides 3 commands:
- `tasks`
- `kanban`
- `idea`

More details can be found in the [documentation.](https://absorb.readthedocs.io)
